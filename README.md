# Dodge the spiders 
_This game was made with [Godot](https://godotengine.org/)_

Become a cute little roach and doge the spiders coming at you.

Try to stay alive for the longest time!


Try it on [itch.io](https://nadtr.itch.io/dodge-the-spiders).
